使用终端对MySQL数据库的基础操作

终端操作数据库

基础操作

  • 登陆数据库
    mysql -u root -p （+ 密码）

  • 查询数据库服务器中所有数据库
    show databases

  • 选用某一个数据库中数据
    use 想用的数据库的名字
    select * from 数据库里面的表;

  • 退出
    exit;

  • 创建数据库
    creat database 名字;

  • 查看某个数据库中所有的数据表
    show tables;

  • 创建数据表
    CREATE TABLE 名字（表头 + 表头类型）;

  • 查看数据表的结构
    describe 名字

  • 删除数据表
    drop table 表名

  • 添加数据记录
    INSERT INTO 表
    VALUES （‘参数’）

  • 删除数据
    delete from 表名 where 其中一个参数（一个参数表示一行数据）

  • 修改数据
    update 表名 set 想要修改的参数 where 修改参数的那一行里的某一个参数

  • 查看默认字符格式
    SHOW VARIABLES LIKE 'character%';

  • 修改默认字符格式

  • 输错命令

    常用数据类型

  • 数值（从上到下，依次范围逐渐增大，常用INT）

    • TINYINT
    • SMALLINT
    • MEDUIUMINT
    • INT & INTEGER
    • BIGINT
    • FLOAT
    • DOUBLE
    • DECIMAL

  • 日期/时间

    • DATE：YYYY-MM-DD
    • TIEM：HH:MM:SS
    • YEAR：YYYY
    • DATETIME：YYYY-MM-DD HH:MM:SS
    • TIMESTAMP：YYYYMMDD HHMMSS

  • 字符串

    • CHAR：定长字符串
    • VARCHAR：变长字符串
    • TINYBLOB：不超过 255 个字符的二进制字符串
    • TINYTEXT：短文本字符串
    • BLOB：二进制形式的长文本数据
    • TEXT：长文本数据
    • MEDIUMBLOB：二进制形式的中等长度文本数据
    • MEDIUMTEXT：中等长度文本数据
    • LONGBLOB：二进制形式的极大文本数据
    • LONGTEXT：极大文本数据

  • 注意：
    日期按照格式，数值和字符串按照大小
